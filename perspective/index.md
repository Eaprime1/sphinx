# Perspective Documentation

Perspective is a living knowledge architecture — a system for organizing, routing, and
transforming information through gravity wells, pools, and the Aperture.

This documentation follows the [Diátaxis](https://diataxis.fr/) framework, structured
into four quadrants: **Tutorials** (learning by doing), **How-to guides** (task recipes),
**Explanations** (understanding the cosmology), and **Reference** (precise definitions).

:::{note}
**AI-first project.** These docs are authored collaboratively by humans and AI partners.
See {doc}`ai-playbook` for the conventions used throughout.
:::
