start to finish. No prior knowledge of the Perspective architecture is assumed.

## Planned tutorials

**From zero to live docs: Sphinx + MyST + Perspective**
: Set up the full documentation stack from scratch. By the end you have a working
  Sphinx build with MyST enabled, the Perspective structure in place, and your first
  page published in all three formats (HTML, ePub, PDF).

**Building your first gravity well doc set: Physical / Mental / Spirit**
: Create a three-well architecture from scratch — one Gravitar for foundational
  knowledge, one Magnetar for active work, one Pulsar for recurring reviews.
  Walk through creating pools, wiring the Aperture, and running your first Sparklization.

**Creating an Aperture between Google Drive pools and Box wells**
: A hands-on walkthrough of connecting a Google Drive pool to a Box gravity well via
  the Aperture governance layer. Covers pressure scoring, ethics flags, and the
  automation trigger setup.
